Project Objectives

Create an algorithm and export it into an API using Google’s Teachable Machine. Reflecting on Safiya Noble's book and this activity, submit a project statement with an embedded video of the algorithm at work.

Project Scope

Classification: Popular/Mainstream Candy Sold in the United States

Range: 10 King Size Candies

Item List: Almond Joy, Crunch, Heath, Kit Kat, M&Ms, Reese's, Snickers, Starburst, Twix, Wuchamacallit

Image Samples: Over 300 for each classified item

Process

Initially, only 150 samples were collected for each item. The machine model was not able to differentiate and identify over half of the candy bars. The sample intake was increased to at least 300 for each item.

Overall, the machine model was able to identify all candy bars but had some issues with Twix Bars. This may be due to sample item quality and variety.

Lessons Learned

This project demonstrated that code is nuanced, complex, and requires validation. Many factors and variables must be considered to ensure the model is reliable and efficient.

Applying these lessons showed the importance of considering social, cultural, and power dynamics, as mentioned by Safiya Noble in her book.
